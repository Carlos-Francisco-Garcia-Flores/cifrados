Cifrar/Descifrar con Cifrado Escítala

Instrucciones:

El Cifrado Escítala utiliza un método de transposición en el que se organiza el mensaje en un número determinado de columnas. Para utilizarlo:

  • Ingresa el mensaje que deseas cifrar o descifrar.
  • Introduce el número de columnas, que debe ser un número entero positivo.
  • Selecciona si deseas cifrar o descifrar el mensaje.
  • Haz clic en Ejecutar para ver el resultado.

Cifrar/Descifrar con Cifrado César

Instrucciones:

El Cifrado César funciona desplazando cada letra de tu mensaje un número determinado de posiciones en el alfabeto. Para utilizarlo:

  • Ingresa el mensaje que deseas cifrar o descifrar.
  • Introduce un número de desplazamiento. Este número debe ser entero y positivo.
  • Selecciona si deseas cifrar o descifrar el mensaje.
  • Haz clic en Ejecutar para ver el resultado.
